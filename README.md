# XCSkin-Android Common UI Lib   
 
1.自己开发的Android App中常用的自定义UI库,主要包括了如下常用UI:
2.圆环百分比进度条控件
3.水平百分比进度条控件
4.下拉列表框框架
5.广告轮播图控件
6.滚动条（仿ViewPager效果）控件
7.和圆角矩形ImageView控件
8.圆环进度条控件
9.侧滑菜单控件
10.球形水波纹上升进度控件
11.刮刮卡效果控件
12.卫星式菜单控件
13.热门标签控件
14.仿微信动态圆饼图控件，底部弹窗控件
