# XCSkin-Android Common UI Lib  
自己开发的Android App中常用的自定义UI库,主要包括了如下常用UI:

圆环百分比进度条控件，  水平百分比进度条控件

下拉列表框框架，广告轮播图控件

水平滚动条（仿ViewPager效果）控件 ，            圆形和圆角矩形ImageView控件

开口圆环进度条控件 ，                           侧滑菜单控件

球形水波纹上升进度控件 ，                     刮刮卡效果控件

卫星式菜单控件，热门标签控件

仿微信动态圆饼图控件，底部弹窗控件
